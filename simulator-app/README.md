# 🔬 Physics Simulator

Interactive web application for exploring physics through real-time simulations.

## ✨ Features

### 🚀 Projectile Motion
- Adjustable launch angle (0-90°)
- Variable initial velocity
- Custom gravity settings
- Real-time trajectory visualization
- Height, distance, and time calculations
- Velocity vector display

### ⏱️ Simple Pendulum
- Adjustable mass and length
- Damping control
- Real-time oscillation visualization
- Period calculation
- Energy demonstration

### 🌍 Planetary Orbit
- Multiple planet selection (Mercury, Venus, Earth, Mars, Jupiter)
- Variable simulation speed
- Orbital mechanics visualization
- Real planetary data
- Distance, velocity, and period display

## 🛠️ Technologies

- **HTML5** - Semantic structure
- **CSS3** - Modern styling with CSS variables
- **JavaScript** - Interactive functionality
- **Canvas API** - Real-time rendering
- **Font Awesome** - Professional icons
- **Google Fonts** - Inter typography

## 🚀 Quick Start

### Use Locally:
```bash
python -m http.server 8000
# Visit http://localhost:8000/simulator-app
```

Or simply open `index.html` in your browser.

## 📖 How to Use

### Projectile Motion
1. Select angle, velocity, and gravity
2. Click "Launch" to start simulation
3. Watch the trajectory and real-time data
4. Use "Reset" to clear and start over

### Simple Pendulum
1. Adjust mass, length, and damping
2. Click "Start" to begin oscillation
3. Observe the motion and period
4. Use "Reset" to stop and reset

### Planetary Orbit
1. Select a planet from the dropdown
2. Adjust simulation speed
3. Click "Start" to begin orbit
4. Use "Pause" to pause/resume
5. Use "Reset" to restart
